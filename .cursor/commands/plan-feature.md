Follow the workflow defined in `.github/prompts/plan-feature.prompt.md`.

This command plans a new feature with strategic analysis. It will:
- Analyze problem and user needs
- Define user stories and value
- Create acceptance criteria
- Assess technical scope
- Identify dependencies
- Create feature file
- Add to ROADMAP.md

**Usage**: Describe the feature you want to plan, or it will prompt you for details.

**See**: `.github/prompts/plan-feature.prompt.md` for complete workflow details.
